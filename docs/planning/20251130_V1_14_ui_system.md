# Task 14: UI System (HUD)

## Objective
Implement a Heads-Up Display (HUD) to show game status.

## Context
The player needs to see their current resources, wave number, and health of the Kobayashi Maru.

## Requirements

### 1. UI Overlay (HTML/CSS)
- Create a DOM overlay on top of the canvas in `index.html`.
- **Elements:**
  - Resource Counter: "Matter: 500"
  - Wave Counter: "Wave: 1"
  - Health Bar: "Integrity: 100%"
  - Turret Selection Toolbar: Buttons to select turret types.

### 2. UI Manager (`src/ui/UIManager.ts`)
- **Methods:**
  - `updateResources(amount)`: Update text.
  - `updateWave(wave)`: Update text.
  - `updateHealth(percent)`: Update health bar width/color.
  - `initialize()`: Bind buttons to `PlacementManager`.

### 3. Integration
- Call UI updates from `ResourceManager`, `WaveManager`, and `Game` loop.

## Acceptance Criteria
- [ ] HUD is visible over the game.
- [ ] Resource count updates in real-time.
- [ ] Wave number updates.
- [ ] Health bar reflects Kobayashi Maru status.
- [ ] Turret selection buttons work.

## Files to Create/Modify
- `src/ui/UIManager.ts`
- `index.html`
- `src/style.css`
